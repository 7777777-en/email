# Email Sysytem 

### Web-based application using Python, Django and JavaScript


#### Table of contents:
* User management system
* Inbox page
* Compose new email
* View email detail
* Archive or Unarchive emails
* Reply to emails

### Technologies
This project is created with:
* JavaScript
* JSON
* Django Framework
* Python
* SQL
* Bootstrap
* HTML
* CSS
